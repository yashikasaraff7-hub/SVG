# SVG
This project demonstrates the use of SVG (Scalable Vector Graphics) in HTML to create different shapes, text elements, animations, and gradient effects. The main objective of this code is to showcase how vector graphics can be drawn directly inside a webpage without using images.

The webpage includes basic shapes such as circles, rectangles, lines, polygons, and complex paths. It also demonstrates animation techniques like movement, scaling, color change, opacity transition, and stroke drawing effects.

This project highlights the power of SVG in creating scalable, interactive, and animated graphics using only HTML elements without any external libraries or images.
Technology Used
	•	HTML5
	•	SVG (Scalable Vector Graphics)
	•	Built-in SVG Animation (<animate> and <animateTransform>)
  This is a complete SVG practice project that covers basic shapes, advanced paths, animations, and gradient effects. It shows how powerful and flexible SVG is for creating scalable and animated web graphics.
